Proxy SwitchyOmega教程，添加规则设置，配合v2rayN和Clash使用方法

Proxy SwitchyOmega使用教程：▶ https://youtu.be/OvTHi5TRWMc

一、Proxy SwitchyOmega 是什么？
Proxy SwitchyOmega 是一款浏览器代理扩展程序，可以轻松切换多个代理设置，支持Chrome和Firefox等浏览器使用。可以只让浏览器使用代理，电脑上的其它软件不走代理。

二、Proxy SwitchyOmega 下载与安装
1、Chrome 或基于 Chromium 的浏览器
在线安装：从 Chrome 应用商店 安装
离线安装：去 Github 下载 安装包，选择扩展名为 .crx，或者使用备用下载>>

2、Firefox 或基于 Mozilla 的浏览器
在线安装：从 Mozilla Add-Ons 安装
离线安装：去 Github 下载 安装包，选择扩展名为 .xpi，或者使用备用下载>>

三、Proxy SwitchyOmega 使用教程
首先需要添加 HTTP/SOCKS 代理服务器
可以用 Clash 或 v2rayN 软件将机场节点转成SOCKS节点使用。

情景模式： auto switch
可根据 规则列表 代理网站，和Clash的“规则模式”或v2rayN“绕过大陆模式”类似。支持自定义规则哪些网站是否需要走代理。
规则列表：gfwlist [Github]，gfwlist [备用]
添加条件：可以将网址单独添加到“切换规则”里，选择是否走代理。
*Clash和v2rayN代理软件本身也支持分流。
